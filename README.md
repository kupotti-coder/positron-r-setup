> 画面のどこをクリックすればいいかを、ひとつずつ丁寧に書きました。

## 🎯 この記事でやること（全体の流れ）

1. **R をインストール**（計算の「エンジン」）
2. **Positron をインストール**（コードを書く「作業場」）
3. **Positron を起動して、初めてのコード `print("Hello World!")` を実行**
4. **拡張機能を追加**（日本語化＋AIアシスタント「Continue」）
5. **Continue にGeminiの鍵を設定**
6. **AIにRコードを書いてもらって実行してみる**

> 🚗 **イメージ：** R は「車のエンジン」、Positron は「運転席（ハンドルやメーターがある場所）」です。
> エンジンだけでは運転できないので、**必ず R を先にインストール**します。

---

## STEP 1：R をインストールする

### 1-1. ダウンロードページを開く

下のリンクを開きます。

👉 https://cloud.r-project.org/

### 1-2. 自分のパソコンに合うものを選ぶ

ページ上部に3つのリンクがあります。**自分のパソコンのOS**を選んでください。

- Windows の人 → **「Download R for Windows」**
- Mac の人 → **「Download R for macOS」**
- Linux の人 → **「Download R for Linux」**

### 1-3.（Windowsの場合）インストーラをダウンロード

1. 「Download R for Windows」をクリック
2. 次のページで **「base」** をクリック
3. **「Download R-4.6.x for Windows」** という青いリンクをクリック
4. ダウンロードされた `R-4.6.x-win.exe` をダブルクリック

### 1-4. インストーラを進める

- 言語を聞かれたら「日本語」または「English」でOK
- あとは基本的に **「次へ（Next）」を押し続けるだけ** で大丈夫です（設定はデフォルトのままで問題ありません）
- 最後に「完了（Finish）」を押せばインストール完了です

> 🍎 **Mac の人へ：** ダウンロードした `.pkg` ファイルをダブルクリックし、指示に従って進めます。
> もし「開発元を確認できないため開けません」と出たら、`システム設定 → プライバシーとセキュリティ` を開き、「このまま開く」を押してください。

> ✅ **ここでは「Rが入った」ことだけ確認できればOK。** Rを直接起動して使うことはありません。次のPositronから呼び出して使います。

---

## STEP 2：Positron をインストールする

### 2-1. ダウンロードページを開く

👉 https://positron.posit.co/download.html

### 2-2. 規約に同意してダウンロード

1. ページにある **チェックボックス**（ライセンスとプライバシーポリシーへの同意）にチェックを入れます
2. 自分のOS用のダウンロードボタンを押します
   - Windows → `.exe` ファイル
   - Mac → `.dmg` ファイル

### 2-3. インストールする

- **Windows：** ダウンロードした `.exe` をダブルクリックし、「次へ」を押し続けます
- **Mac：** `.dmg` を開いて、Positron のアイコンを **「Applications（アプリケーション）」フォルダにドラッグ** します

> 💡 Positron は一度入れれば、**自動で最新版に更新**してくれます。便利！

---

## STEP 3：Positron を起動して「Hello World!」を実行する

ここからが本番です。ゆっくりやれば大丈夫。

### 3-1. Positron を起動する

インストールした Positron をダブルクリックして起動します。
（初回起動時はテーマ選択や説明が出ることがありますが、そのまま進めてOKです）

### 3-2. 作業用フォルダを用意する（Cドライブ直下に `zemi`）

まず、コードを保存する「フォルダ（部屋）」を作ります。

1. **エクスプローラー（Windowsのファイル管理画面）** を開く
2. **Cドライブ（`C:`）を開く**
3. 何もないところで右クリック →「新規作成」→「フォルダー」
4. 名前を **`zemi`** にする

これで `C:\zemi` というフォルダができました。

### 3-3. Positron で、そのフォルダを開く

1. Positron の画面左上のメニュー **「File（ファイル）」** をクリック
2. **「Open Folder...（フォルダーを開く）」** をクリック
3. さっき作った **`C:\zemi`** を選んで「開く」
4. 「このフォルダー内のファイルの作成者を信頼しますか？」と聞かれたら **「はい、作成者を信頼します（Yes, I trust the authors）」** を選びます

> 📁 これで Positron が「`zemi` という部屋」で作業する状態になりました。
> 画面の左側に `ZEMI` というフォルダ名が表示されていればOKです。

### 3-4. ファイルを作る（`hello.R`）

1. 画面左側の `ZEMI` フォルダ名の右側に出る **「新しいファイル」アイコン**（紙に＋マークのアイコン）をクリック
   - または **「File」→「New File」** でもOK
2. ファイル名を **`hello.R`** と入力してEnter
   - ⚠️ **最後の `.R` を忘れずに！** これが「これはRのコードですよ」という目印になります

すると、右側にまっさらな編集画面が開きます。

### 3-5. コードを書く

開いた `hello.R` の画面に、次の1行を入力します。

```r
print("Hello World!")
```

- `print` … 「画面に表示してね」という命令
- `"Hello World!"` … 表示したい文字（ダブルクォーテーション `" "` で囲むのがルール）

### 3-6. 実行する！

書いたコードの行にカーソルを置いた状態で、キーボードの

- **Windows：`Ctrl` + `Enter`**
- **Mac：`Cmd（⌘）` + `Enter`**

を押します。

> 💬 初回実行時は「Rを起動しています…」と少し時間がかかることがあります。落ち着いて待ちましょう。

画面下のほうにある **「Console（コンソール）」** に、こう表示されたら **大成功！** 🎉

```
[1] "Hello World!"
```

> 🎊 **おめでとうございます！** あなたは今、人生で初めてプログラムを実行しました。
> `[1]` という数字は「1個目の結果ですよ」という意味なので、気にしなくて大丈夫です。

## STEP 4：拡張機能（エクステンション）を追加する

Positron はそのままでも使えますが、**「拡張機能」** を入れると、画面を日本語にしたり、AIに手伝ってもらったりできるようになります。

拡張機能は、画面左側にある **四角が4つ集まったようなアイコン（Extensions）** から追加します。

### 4-1. 日本語化する（日本語パッチを入れる）

1. 左側の **Extensions（拡張機能）アイコン** をクリック
2. 上の検索ボックスに **`Japanese`** と入力
3. **「Japanese Language Pack for Visual Studio Code」**（日本語パック）が出てくるので、**「Install（インストール）」** をクリック
4. インストール後、画面右下に「再起動しますか？（Restart / Change Language）」のような通知が出るので、**再起動（Restart）** を選びます
   - 通知が出なかった場合：`Ctrl + Shift + P`（Macは `Cmd + Shift + P`）を押し、出てきた入力欄に `Display Language` と打って **「Configure Display Language」→「日本語」** を選び、Positronを再起動します

再起動後、メニューが日本語になっていれば成功です。

### 4-2. AIアシスタント「Continue」を入れる（手動インストール）

「Continue」は、AIにコードを書いてもらえる拡張機能です。
**Positronでは通常の検索からは入れられないことがあり、その場合は手動でインストール**します。

#### 手順A：まず検索で探してみる

1. Extensions（拡張機能）の検索ボックスに **`Continue`** と入力
2. もし **「Continue」** が表示されて「Install」できるなら、それでインストール完了です（次のSTEP 5へ）

#### 手順B：見つからない／入れられない場合（手動インストール）

1. ブラウザで Continue の配布ページを開き、拡張機能ファイル（**`.vsix` という拡張子のファイル**）をダウンロードします
   - 「Continue vsix download」などで検索すると、配布元（Open VSX や GitHubのReleasesページ）が見つかります
2. Positron に戻ります
3. Extensions（拡張機能）画面の **右上にある「…（三点リーダー）」メニュー** をクリック
4. **「Install from VSIX...（VSIXからインストール）」** を選択
5. さっきダウンロードした **`.vsix` ファイルを選んで「インストール」**

インストールが終わると、**画面左側に少し傾いた六角形（テトリスのL字ブロックを3つ組み合わせたような形）のアイコン** が増えます。これがContinueです。

> 💡 **ちょっと豆知識：** Positronには「Posit Assistant」という純正のAI機能も搭載され始めています。
> ただ今回は、Geminiを無料で使える「Continue」で進めます。

---

## STEP 5：Continue で Gemini を使えるようにする

Continueに「AIの頭脳」を接続します。今回はGoogleのAI「**Gemini**」を使います。
そのためには **「APIキー（API鍵）」** という、いわば“専用の合鍵”が必要です。無料で取得できます。

### 5-1. Google AI Studio で鍵（APIキー）を取得する

1. 下のページを開きます（Googleアカウントでログインしてください）

   👉 https://aistudio.google.com/prompts/new_chat

2. 左側または上部のメニューから **「Dashboard」**（または「API keys」）を開きます
3. **「APIキーを作成（Create API key）」** をクリック
4. プロジェクトを作成します。今回は分かりやすいように：
   - **プロジェクト名：`Positron`**
   - **キー名：`Positron Key`**
   として作成します
5. 作成された **APIキー（長い英数字の文字列）を「コピー」** しておきます

> 🔑 **APIキーはパスワードと同じくらい大事**です。他人に教えたり、SNSに貼ったりしないように！
> もし画面を閉じてしまっても、同じ「Dashboard」から確認・再コピーできます。

### 5-2. Continue にキーを貼り付ける

1. Positron に戻り、左側の **Continue アイコン（傾いた六角形）** をクリック
2. **「Google Gemini」** の項目にある **「Enter your Google Gemini API key（Gemini APIキーを入力）」** という欄に、
3. さっきコピーした **APIキーを貼り付け（ペースト）** ます
   - Windows：`Ctrl + V` ／ Mac：`Cmd + V`

### 5-3. 使うモデルを「Gemini 3 Flash Preview」に設定する

1. Continueの **右上にある「設定（歯車⚙️）ボタン」** をクリックします
   - もし「画面が狭すぎる」というような表示が出たら、Continueのパネルの境界を**ドラッグして横幅を広げて**ください
2. 設定画面の **左側メニュー「Models（モデル）」** をクリック
3. **「Chat（チャット）」** と **「Edit（編集）」** の両方を、**`Gemini 3 Flash Preview`** に設定します
   - ※一覧に `Gemini 3.5 Flash` などより新しいモデルがあれば、そちらを選んでもOKです

これで、AIに質問してコードを書いてもらう準備が整いました！

---

## STEP 6：AIにRコードを書いてもらって実行する

いよいよ、AIにコードを書いてもらいましょう。

### 6-1. AIにお願いする

Continueのチャット欄に、たとえば次のように入力して送信します。

> **100個の乱数を作成して、文字化けしないように、ヒストグラムを描くRコード**

- 「乱数（らんすう）」＝ランダムな数字
- 「ヒストグラム」＝データの分布を棒グラフのように表したグラフ
- 「文字化けしないように」＝グラフの日本語が□□□と化けないようにする、というお願い（Rは日本語表示でつまずきやすいので、こう頼むと安心です）

### 6-2. 出てきたコードを取り込む

AIがいくつかコードを提案してくれます。

1. 気に入ったコードのブロックを見つけます
2. そのコードの上にある **「Insert code（コードを挿入）」ボタン**（四角に左向きの矢印が描かれたアイコン）をクリック
3. すると、いま開いているRファイル（`hello.R` など）にコードが貼り付けられます

### 6-3. 実行する

貼り付けられたコードを選択して、STEP 3と同じように

- **Windows：`Ctrl + Enter`**
- **Mac：`Cmd + Enter`**

で実行します。

うまくいくと、画面右側の **「Plots（プロット）」** という場所に、ヒストグラム（グラフ）が表示されます！📊

> 🎉 ここまでできたら、あなたはもう「RでAIを使ってデータ分析を始められる人」です。

## ❓ うまくいかないときは（よくあるつまずき）

| 症状 | 確認すること |
|---|---|
| `print` を実行しても何も出ない | カーソルがコードの行にあるか確認。`Ctrl + Enter`（Macは `Cmd + Enter`）を押せているか確認 |
| 「R interpreter が見つからない」と出る | STEP 1のRが正しくインストールされているか確認。Positronを一度再起動してみる |
| グラフの日本語が「□□□」になる | AIへの指示に「文字化けしないように」と入れて、コードを作り直してもらう |
| Continueが反応しない／エラーになる | APIキーが正しく貼り付けられているか、モデルが選択されているか（STEP 5）を再確認 |
| メニューが英語のまま | STEP 4-1の日本語化のあと、Positronを**再起動**したか確認 |
